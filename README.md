# AI-Diet_Recommendation-System
Machine Learning based diet recommendation system using Python and Flask
AI-Based Personalized Diet Recommendation System

This project predicts a person's body condition and recommends a diet plan using Machine Learning.

Technologies Used

- Python
- Pandas
- Scikit-learn
- Flask
- HTML/CSS

Features

- Predict BMI category
- Calculate BMI
- Daily calorie requirement calculation
- Personalized diet suggestion

How to Run

1. Install requirements:
pip install pandas scikit-learn flask

2. Train model:
python model.py

3. Run application:
python app.py

4. Open browser:
http://127.0.0.1:5000
